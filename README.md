# <a href="https://anassabbou.github.io/Keeper-App/">Keeper</a>

Keeper App is a clone of Google Keep, developed using React, JavaScript, Bootstrap, and Vite.

## Features

- **Note-taking**: Create and delete notes.
- **Responsive Design**: Works seamlessly across devices.
